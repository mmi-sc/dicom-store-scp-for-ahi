# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability related to this project, please report it privately.

Please do not open a public GitHub issue for security vulnerabilities.

Instead, contact the project maintainers at:

aws-sales@mmi-sc.co.jp

We will review the report and respond as appropriate.

## Scope

This repository provides software that is deployed and operated within a customer's AWS account.

The vendor does not operate or manage customer AWS environments.

Customers are responsible for:

- operating their AWS infrastructure
- configuring IAM policies and network security
- monitoring and maintaining their environments

## Updates

Security fixes may be provided through software updates or repository releases when applicable.

Customers are responsible for applying updates within their own environments.
